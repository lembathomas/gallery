
# Gallery App

A cloud-ready photo gallery web application built with Node.js, Express, MongoDB, and deployed using Render. This project includes CI/CD integration using Jenkins with Slack notifications.

## Features

- Upload, view, and manage images  
- Dynamic landing page  
- MongoDB Atlas integration  
- CI/CD pipeline via Jenkins  
- Slack notifications on successful deploy  

---

## Tech Stack

- **Backend:** Node.js, Express  
- **Database:** MongoDB Atlas  
- **View Engine:** EJS  
- **CI/CD:** Jenkins  
- **Hosting:** Render  
- **Notifications:** Slack
